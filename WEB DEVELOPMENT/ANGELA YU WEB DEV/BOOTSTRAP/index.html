I wanted to expand my knowledge, so I turned to YouTube to learn Express.js, Node.js, and MongoDB to create my own REST API, which powers my portfolio. The entire portfolio has been designed by me using Figma.

Regarding my soft skills, I'm creative, an autodidact, and independent.
